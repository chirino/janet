General Information

Java Native Extensions (JANET) is a programming tool simplifying integration of native code with Java applications. A foundation for such integration is provided by the Java Native Interface (JNI) API. However, that low-level API exhibits bare bones of the Java Virtual Machine, and as such it is very inconvenient to use and error-prone. JANET simplifies development process by defining a lightweight Java language extension enabling work on a much higher level of abstraction, closer to that of the Java language itself. JANET translation tool is used to generate ordinary JNI code out of JANET source files. Detailed project info and documentation can be found at the project home page. Please also see release notes for installation and usage instructions.
